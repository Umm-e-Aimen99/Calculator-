#   Basic Calculator App

## Project Overview

The Basic Calculator App is a simple web application developed using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. This project was created to practice JavaScript fundamentals and improve my understanding of how JavaScript interacts with HTML elements.

## Features

- Perform Addition
- Perform Subtraction
- Perform Multiplication
- Perform Division 
- Displays the calculated result instantly
- Prevents division by zero
- Simple and responsive user interface
- Beginner-friendly JavaScript implementation


## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click "index.html" or open it using any modern web browser.
4. Enter two numbers.
5. Click any operation button (+, -, X, ÷) to view the result.

## How the Calculator Works

1. The user enters two numbers in the input fields.
2. JavaScript reads the values entered by the user.
3. The selected operation button triggers a function.
4. The calculation is performed.
5. The result is displayed immediately on the webpage.
6. If the user tries to divide by zero, an error message is shown instead of a result.

## Challenges Faced

One challenge I faced was that input values were treated as strings instead of numbers. This caused incorrect results during addition. I solved this problem by using the `Number()` function to convert the input values before performing calculations.

Another challenge was handling division by zero. I used an `if` statement to check the second number before performing the division.
